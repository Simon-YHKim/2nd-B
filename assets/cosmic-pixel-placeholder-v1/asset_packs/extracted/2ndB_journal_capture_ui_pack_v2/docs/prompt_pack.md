# Prompt Pack — Journal / Capture Copy and Summaries

## Shard title prompt
Given a user journal/capture input, produce a short title in Korean, 12 characters to 22 characters when possible. Do not diagnose or evaluate the user. Keep it warm and concrete.

## Shard preview prompt
Summarize the saved shard in one sentence. Mention what the shard is about, not what it says about the user as a person.

## Connection suggestion prompt
Suggest possible graph node connections using only available user records and existing nodes. Return candidates with confidence and reason. Avoid fixed personality judgments.
