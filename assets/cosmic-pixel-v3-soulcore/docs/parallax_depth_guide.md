# Parallax / Depth Guide

No blur. Depth is expressed only with scale, desaturation via token choice, and opacity.

| Layer | Scale | Token treatment | Opacity | Motion rate |
|---|---:|---|---:|---:|
| L0 deep space | 1.00 | `space950` + sparse `mistGray` stars | 1.00 | 0.10× |
| L1 far Pattern Cores | 0.74–0.82 | `space700` strokes, accent reduced | 0.34–0.50 | 0.28× |
| L2 mid Pattern Data | 0.88–0.96 | `space800` body, accent visible | 0.62–0.78 | 0.55× |
| L3 near active core | 1.00–1.12 | full accent + `moonWhite` highlights | 0.90–1.00 | 1.00× |
| L4 focus / pulse | 1.05–1.18 | `soulViolet` / `signalMint` glow | 0.22–0.64 | 1.15× |
