① 이미 API 연결된 것

작동/구성 확인됨
키/토큰이 실제로 설정돼 호출이 성립하는 시스템. LLM 키는 모두 Supabase Edge Function 뒤 서버사이드로 숨겨져 있어(클라이언트 번들 노출 0), 레포에서는 프록시 함수의 존재만 확인 가능하고 키 자체는 Supabase secret에 있음.
SupabaseCONNECTED
Postgres DB · Auth · Edge Functions · Storage
URL+anon 키가 eas.json의 preview/production 프로필에 박혀 있음(레포에서 직접 확인). 데이터·인증·서버리스 함수의 중추.

② API가 꼭 필요한 시스템 (코어)

없으면 앱 작동 불가
제품이 최소 기능을 하려면 반드시 있어야 하는 키. 셋 다 이미 채워져 있음 — 즉 출시에 필요한 필수 API는 현재 모두 충족.
Supabase필수 · 충족✓
데이터 영속화 · 로그인 없으면 빈 껍데기
유저 데이터·자기이해 기록·ai_audit_log(C3) 전부 여기 의존.
Google AI (Gemini)필수 · 충족✓
개인 비서/대화의 두뇌
키는 gemini-proxy(Supabase secret) 경유. 키 만료·쿼터 초과 시 핵심 경험 정지. 프로덕션엔 Vertex AI 전환(아래 ③) 권장.
EXPO_TOKEN필수 · 충족✓
빌드 배포 + OTA 발행 자동화
없으면 폰에 업데이트 도달 불가(매번 수동 빌드). 등록으로 해결됨.

③ API 있으면 더 좋아지는 것

선택 · 켜면 기능 활성화
코드엔 이미 자리(EXPO_PUBLIC_* 키)가 있지만 값이 비어 해당 기능이 꺼져 있는 상태. eas.json에 이들 값은 0건 — 키만 넣으면(일부는 리빌드 필요) 켜짐.
Sentry미설정
에러 모니터링
현재 SDK는 @sentry/browser = 웹 전용. DSN만 넣으면 웹 에러 리포팅 즉시 가동(리빌드 X). 네이티브 Android 크래시까지 잡으려면 @sentry/react-native로 교체 + EAS 리빌드 필요 — docs/sentry-setup.md 참고.

"알아가기→비서→구체화" 3축 중 어디서 이탈하는지 측정. XPRIZE 임팩트 증빙에 유용. env.ts 슬롯 존재(.optional).

④ 에이전트(Claude Code)에 연결된 통합

개발/운영 워크플로우
제품이 아니라 Simon의 개발·운영을 돕는 도구 쪽 API. Claude Code 세션에 연결된 MCP 통합 — 코드/디자인/문서/배포를 한 자리에서.
GitHub Supabase Vercel Figma Notion Slack Gmail Google Drive Google Calendar Canva Linear Context7 (라이브러리 문서) Hugging Face Wolfram Bitly Spotify Melon
개발에 직접 쓰이는 핵심: GitHub·Supabase·Vercel·Figma·Context7. 나머지는 보조/개인용. 제품 안정성을 위해 추가하면 좋은 MCP: Sentry(③의 모니터링과 짝).

출시(2026-08-17) 전 추천 우선순위

  1. Sentry DSN 연결(웹) — DSN만 넣으면 웹 에러 리포팅 즉시 가동, 무료, 리빌드 불필요. 가장 ROI 높음. 네이티브 크래시는 별도 리빌드.
  2. Kakao/Google 소셜 로그인 — 한국 타깃 가입 마찰 제거. 전환율 직결.
  3. PostHog 또는 Clarity — 3축 퍼널 측정 = XPRIZE 임팩트 증빙 데이터.
  4. Vertex AI 전환 — 트래픽 늘기 전, 쿼터·과금 안정화(C2).
  5. RevenueCat — 수익화 시작 시점에. 출시 직후엔 후순위 OK.
※ "미설정"은 코드에 키 슬롯(EXPO_PUBLIC_*)은 있으나 값이 비어 기능이 꺼진 상태 — 키만 넣으면 켜짐(일부는 EAS 리빌드 필요). LLM 키(Gemini/Claude)·EXPO_TOKEN은 레포 파일이 아니라 Supabase secret / GitHub Actions secret에 존재하므로, 본 문서의 해당 "연결됨" 판정은 CI·런 로그·프록시 함수 존재에 근거함. · 2nd-Brain · Gemini XPRIZE 출품작
