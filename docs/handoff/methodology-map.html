A.3 추론 엔진 — "Inference Engine v1" persona/build.ts

원자료(records/sources) → buildPersona() → PersonaCard. 파이프라인:
  • ① 휴리스틱 스코어 scoreFromAnswers — KO/EN 이중 정규식 키워드. 개방성=길이구동(0.4+avgLen/600), 그 외 0.4+hits×0.12 (cap .95). interview 트랜스크립트 제외(길이 휴리스틱 saturation 방지)
  • ② BFI override — BFI-44 기록 있으면 휴리스틱 대체, 1–5→0–1 정규화. "BFI는 신경성 직접측정 → 반전 불필요(구 TIPI의 emotional_stability와 달리)". traitsSource: heuristic→bfi
  • ③ 신뢰도 SOKA traitConfidenceFor — BFI→{questionnaire, high}; journal_text는 관측수 스케일(≥15 high·≥5 medium·else low); 0→{default, low}. anti-Barnum "N개 기록 · medium confidence"
  • ④ deriveValues — 사용자가 실제 답한 프레임워크를 답변수로 랭크
  • ⑤ memorized_patterns 히스토그램 top-3 → 패턴
  • ⑥ LLM 내러티브 요약 callGemini(persona_chat) — 기록 있을 때만 게이트(data-truth/anti-Barnum, C9+C3 유지)

⚠️ build.ts 헤더 1–11행에 아직 "TIPI", "emotional_stability ↔ neuroticism 반전", 로더명 loadLatestBfi — TIPI→BFI 마이그레이션 주석 잔재(실제 경로는 BFI-44). 정리 후보. "Sprint 3 풀 LLM 추출(≥50 기록)"은 아직 미착륙.
