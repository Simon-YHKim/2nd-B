3 · 화면 연결 재설계 포인트 (가장 중요)

고아 화면 (다른 화면에서 들어오는 링크가 없음 — 직접 URL/탭으로만 도달)

/insights · /ops · /esm — 축 2의 핵심 가치(패턴 종합·루틴·체크인)인데 어느 화면도 연결 안 함. 진입점 필요 (예: /profile 퀵칩, /secondb 퀵액션, 또는 /core-brain center card).
/data · /support · /permissions · /manual — 유틸. /settings·/profile 어딘가에서 한 번씩은 닿게.
/iden — 방금 /data에 카드 추가했지만, /data 자체가 고아라 체인이 끊김. /profile 또는 /settings에서 /data(또는 /iden) 직접 진입점 추가 권장.

막다른 화면 (나가는 CTA 없음 — BackArrow로만 탈출)

/interview · /formats · /iden · /insights · /research · /ops · /esm · /support · /plans

완료 후 "다음 한 걸음" CTA가 비어 있음. 예: /interview·/audit 완료 → /persona 또는 /core-brain; /insights → /secondb{관련 패턴}; /research → /secondb 또는 /persona.
