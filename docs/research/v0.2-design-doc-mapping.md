## §12 Round 2-4 deeper backing for existing sections

After round 1, additional research was curated to strengthen specific v0.2 claims:

### §02 Layer 1 — confidence field calibration (NEW backing)

`methodology/uncertainty-calibration.md` (13 refs) establishes:
- Verbalized confidence_pct (Tian 2023 EMNLP) as the primary signal for Gemini
- SelfCheckGPT (Manakul 2023) as secondary at Golden Set time only (cost control)
- Release-gate thresholds: Brier ≤ 0.20 / ECE ≤ 0.10 marginal / ECE ≤ 0.15 per subgroup / insufficient-tier recall ≥ 0.70 + precision ≥ 0.80
- 9 specific recommendations C1-C9 mapped to v0.2 sections

### §02 Layer 2 — trait notion refinement (NEW backing)

`batches/whole-trait-density.md` (13 DOIs) shifts the paradigm:
- Fleeson 2001 JPSP + Fleeson & Jayawickreme 2015 JRP: trait = density distribution of states
- Implication: Tier A atomic quote MUST include timestamp + situation_tags, not just observation
- DIAMONDS situation taxonomy (Rauthmann 2014) as latent situation-tag schema
- Cold-start thresholds (< 10 / 10-29 / 30+ / 80+ entries) tied to claim strength

### §03 Risk 1 — percentile time-cadence (NEW backing)

`batches/personality-change-dynamics.md` (14 DOIs) gives hard numbers:
- 0.1-0.2 SD per decade naturally (Roberts 2006 big meta, Bleidorn 2022 anchor)
- 0.3-0.5 SD over 3mo with structured digital intervention (Stieger 2021 PNAS RCT)
- Jacobson & Truax 1991 RCI: ~0.5 SD detection floor for short intervals
- **Therefore**: percentile UI should update monthly minimum, not weekly

### §03 Risk 3 — self-selection bias (NEW backing)

`batches/self-report-bias.md` (17 DOIs) provides the full academic basis:
- Paulhus 1984 two-component SDR, BIDR validation
- Vazire 2010 SOKA: self knows N better, others know Intellect better
- Cultural: Locke & Baik 2009 Korean acquiescence; Chen 1995 East Asian midpoint
- Per-entry SDR flagging FORBIDDEN; ≥10-entry aggregation required

### §04 Evidence storage architecture (NEW backing)

`methodology/ai-memory-architecture.md` (18 refs):
- Park 2023 UIST generative agents memory pattern
- MemGPT (Packer 2023) + MemoryBank (Zhong 2024 AAAI) external memory
- Tulving 1972: Tier A = episodic, behavior taxonomy = semantic
- Recommended: Supabase pgvector + HNSW (Malkov & Yashunin 2018 TPAMI)
- Nightly / weekly / monthly consolidation tree
- Right-to-erasure 6-step cascade (honoring constraint C3)
- **NO per-user fine-tuning** (catastrophic forgetting + Carlini 2021 extraction risk)

`methodology/privacy-preserving-ml.md` (25 refs):
- DP-SGD (Abadi 2016 CCS): ε ≤ 8 acceptable / ε ≤ 4 strict for sensitive
- Carlini 2021 extraction → vendor-no-train rule
- Federated learning deferred to v0.3 (Bonawitz 2019 operational cost)
- k-anonymity / ℓ-diversity for Korean PIPA pseudonymization
- Tier A atomic quotes ENCRYPTED AT REST

### §05 Ambiguity Resolution Queue (NEW backing)

`methodology/active-learning-hil.md` (18 refs):
- Settles 2009 AL survey, uncertainty sampling, query-by-committee
- Mehrotra 2016 CHI notification fatigue: 3-soft / 5-hard per session
- McHugh 2012 κ thresholds: κ < 0.6 triggers taxonomy refactor
- Priority function: margin + QBC disagreement + downstream impact − cooldown

`batches/metacognition-introspection.md` (14 DOIs) constrains queue design:
- Nisbett & Wilson 1977: users CANNOT reliably introspect behavior patterns
- Vazire 2010 SOKA gates which trait questions to ask
- **Therefore**: weight values/situations questions over trait questions in Queue
- Trait questions framed as "how do you see yourself" not "what are you"

### §06 Country / culture priority — global coverage (NEW backing)

`batches/cross-cultural-global-south.md` (22 DOIs) completes global coverage:
- Latin American (simpatía Triandis 1984, familismo Sabogal 1987)
- African (Ubuntu Metz 2007, Nsamenang 2006 ontogenesis)

### §05/§06/§07 First-session design (NEW backing)

## §14 Surfaced design tensions (require Simon decision before v0.3)

1. **Single-stage vs two-stage prompt structure** (`methodology/llm-agnostic-design.md` Tam 2024 format tax) — v0.2 implies single-stage structured output; literature suggests reflect → emit splits quality.
2. **Trait point vs density representation** (`batches/whole-trait-density.md` Fleeson lineage) — v0.2 §02 implicitly point-estimate; refinement: every observation needs timestamp + situation_tag.
3. **East-Asian framing transfer to Arab users** (`batches/cross-cultural-global-south.md` San Martin 2018) — naive transfer is empirically wrong; need separate cultural priority path.
4. **Voice claims boundary** (`batches/metacognition-introspection.md`) — Voice MUST NOT claim "we know you better than you know yourself" (Nisbett & Wilson 1977; Vazire 2010 SOKA).
5. **Percentile UI cadence** (`batches/personality-change-dynamics.md` Bleidorn 2022) — current implicit assumption of weekly updates likely too frequent; monthly minimum recommended.
6. **Per-user fine-tuning** (`methodology/ai-memory-architecture.md`) — proposal: do not fine-tune base model on user data (catastrophic forgetting + Carlini extraction risk). Use prompt-only personalization.
7. **EU AI Act framing** (`legal/jurisdiction-compliance.md`) — "self-reflection tool / user-reported reflection structuring" is the safe framing; any "emotion recognition" language risks Annex III §1(c) high-risk classification.
