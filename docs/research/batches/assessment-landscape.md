# Framework: Personality & Character Assessment Landscape

> Batch produced via `docs/research/psychology-handoff.md`. All DOIs verified against Crossref / publisher record (May 2026).
>
> This batch is **comparative**: a catalog of well-known assessments tiered by empirical evidence rigor. It is the answer to "버크만 진단처럼 유명하고 신뢰받는 검사 방법들"인데, "유명함"과 "신뢰성"이 늘 일치하지 않음을 명시한다.

## Empirical Tiering

| Tier | Definition | Examples |
| --- | --- | --- |
| **A** | Strong peer-reviewed validation across multiple independent labs; meta-analyses available; replicable factor structure across cultures | Big Five (NEO/BFI-2), HEXACO, Holland RIASEC, Schwartz Values |
| **B** | Real validation evidence but with significant caveats — proprietary instruments with publisher-controlled studies, or measures with construct debates | HPI, 16PF, MSCEIT / EI ability tests, Birkman (partial) |
| **C** | Famous but weak empirical base — face validity high, predictive validity low; reliability acceptable but construct validity contested | MBTI, Enneagram, DISC, CliftonStrengths |
| **D** | No verified peer-reviewed validation evidence base | Pop typologies (love languages, color personality, animal types) |

Tier C is widely used in HR, coaching, and pop self-help, but should never be presented in 2nd-Brain as a basis for trait extraction or Advisor guidance.

### Tier C (use only as starter conversation, not for inference)

**DISC** and **CliftonStrengths (StrengthsFinder 2.0)** are widely used but most validation lives in publisher technical reports (Gallup, Wiley) rather than independent peer review. No DOI-registered independent validation strong enough to anchor an INSERT row is included.

## Philosophy & Methodology — One-paragraph Each

### Big Five (Tier A)
- **Philosophy**: lexical hypothesis — natural language captures the dimensions that matter for describing personality. Five factors emerged repeatedly across languages and cultures.
- **Method**: factor analysis of trait adjectives → questionnaire instruments (NEO-PI-R, BFI-2) → cross-cultural invariance testing.
- **2nd-Brain fit**: trait extraction backbone. Already in production knowledge base.

### HEXACO (Tier A)
- **Philosophy**: lexical extension of Big Five — adds Honesty-Humility as a sixth factor, splits Agreeableness/Emotionality differently.
- **Method**: same as Big Five but six-factor model; 24 facet scales.
- **2nd-Brain fit**: alternative or complement to Big Five for users where honesty/integrity-relevant dimensions matter (work ethics, partner choice). Not currently in seed; future batch candidate.

### Holland RIASEC (Tier A — career)
- **Philosophy**: vocational interests cluster into 6 types (Realistic, Investigative, Artistic, Social, Enterprising, Conventional); person-environment fit predicts career satisfaction.
- **Method**: interest inventory; hexagonal model of type adjacencies.
- **2nd-Brain fit**: career/work-domain interview questions; "what kind of work fits you" prompts.

### Schwartz Values (Tier A)
- **Philosophy**: universal motivational continuum of human values, organized in 19 (refined model) types around two axes (self-protection vs growth; personal vs social focus).
- **Method**: Portrait Values Questionnaire; cross-cultural validation in 10+ countries.
- **2nd-Brain fit**: values clarification module; "what do you actually prioritize" reflection.

### HPI (Tier B)
- **Philosophy**: socioanalytic theory — personality is what others observe (the "reputation" side), used to predict job performance.
- **Method**: 7 primary scales aligned with Big Five but reframed for workplace; meta-analytic validities for job performance (.25–.43).
- **2nd-Brain fit**: limited — Hogan is proprietary and workplace-focused. Useful as conceptual lens for "how others see you vs how you see yourself."

### 16PF (Tier B)
- **Philosophy**: 16 primary "source traits" derived via factor analysis (Cattell) plus 5 global second-order factors.
- **Method**: 185-item questionnaire; internal consistency 0.68–0.87.
- **2nd-Brain fit**: dated but historically influential. Big Five replaces most use cases.

### MSCEIT / Ability EI (Tier B)
- **Philosophy**: emotional intelligence as a set of mental abilities (perceive, use, understand, manage emotions), measurable like cognitive ability via maximum-performance tasks.
- **Method**: 141-item ability test; performance scored against expert/consensus criteria.
- **2nd-Brain fit**: useful conceptual framework for "emotion skills" prompts. Performance test format hard to deliver via journaling — use as vocabulary, not measure.

### Birkman Method (Tier B / proprietary)
- **Philosophy**: distinguishes "usual behavior" (productive style), "needs" (preferred environment), and "stress behavior" (when needs unmet). Each can differ dramatically per individual.
- **Method**: 298-item questionnaire; reports across 9 components and 11 occupational interests.
- **2nd-Brain fit**: the **usual-vs-need-vs-stress** trichotomy is genuinely useful (most assessments collapse these). Apply as conceptual framing in prompts ("how do you act when things are fine" vs "what do you need to keep going" vs "how do you act when those needs aren't met"). **Do not reproduce or reference the proprietary scoring.**

### MBTI (Tier C — caution)
- **Philosophy**: Jungian type theory operationalized into 4 dichotomies (E/I, S/N, T/F, J/P) → 16 types.
- **Method**: forced-choice items; test-retest reliability for type assignment ~50% over 5 weeks (Pittenger 2005).
- **Evidence problems** (Pittenger 2005): no evidence for true type discontinuity (people score continuously on dimensions, not in bimodal clumps); poor predictive validity for job performance; type labels can become self-fulfilling.
- **2nd-Brain fit**: do **not** use for inference. Acceptable as user-stated identifier ("user identifies as INFJ") but do not act on it.

### Enneagram (Tier C — caution)
- **Philosophy**: 9 core types tied to root fears/desires; secondary "wings" and "stress/security arrows."
- **Method**: various questionnaires (RHETI most common); spiritual-tradition origins, not empirical.
- **Evidence problems** (Hook et al. 2021): some reliability but factor analyses rarely recover 9 types; little to no evidence for wings or intertype movement.
- **2nd-Brain fit**: do **not** use for inference. Acceptable as user-stated identifier only.

### DISC (Tier C)
- **Philosophy**: Marston-derived 4-dimension model (Dominance, Influence, Steadiness, Conscientiousness).
- **Method**: forced-choice; multiple proprietary variants (Wiley, others).
- **Evidence problems**: very limited independent peer-reviewed validation; predictive validity for job performance not established at Big Five levels.
- **2nd-Brain fit**: do **not** use.

### CliftonStrengths / StrengthsFinder 2.0 (Tier C)
- **Philosophy**: each person has dominant "talent themes" (34 in the catalog); focus on strengths over weaknesses.
- **Method**: forced-choice items, proprietary scoring (Gallup); validation almost exclusively published in Gallup technical reports.
- **Evidence problems**: weak independent peer-reviewed evidence; instrument design is non-comparative (cannot derive between-person rankings reliably).
- **2nd-Brain fit**: VIA Character Strengths (already in seed) provides similar functionality with stronger evidence base.

## Age Range Coverage

This batch is a meta-analysis; coverage depends on the underlying tool:
- Most assessments target adult populations (18+).
- Youth-validated forms exist for Big Five (BFQ-C), VIA (Youth Survey), and HEXACO (HEXACO-PI-R Junior).
- Career interest measures (RIASEC) extend down to adolescence.

## Application to 2nd-Brain

### Interview Question Examples (validated — assessment-literacy framing)

The 2nd-Brain Advisor should help users **understand their own results** from existing assessments without endorsing or replicating them. Sample prompts:

**Korean**
- 본인이 받아본 성격 검사가 있다면 (예: MBTI, 에니어그램, 버크만, StrengthsFinder), 그 결과가 ''맞다''고 느낀 부분과 ''잘 모르겠다''고 느낀 부분은 각각 어떤 거였어요?
- 그 검사가 본인을 어떤 ''유형''으로 분류했을 때, 그 분류에 본인이 자신을 더 맞추려고 하셨나요, 아니면 ''이건 나랑 안 맞아''라고 거리를 두셨나요?
- 본인은 사람을 ''몇 가지 유형''으로 나눌 수 있다고 생각하시나요, 아니면 모든 사람이 연속적인 스펙트럼 위에 있다고 보세요?

**English**
- If you've taken personality assessments (MBTI, Enneagram, Birkman, StrengthsFinder, etc.), what parts of the result felt accurate, and what parts didn't?
- When that test gave you a "type," did you find yourself leaning into it, or pushing back against it?
- Do you think people fall into a few discrete types, or sit somewhere on a continuous spectrum?

### Trait Extraction Cues

- **Self-typology dependence**: heavy use of test-derived identity language ("as a 4w5", "Activator Achiever", "ESTJ here") in entries. Higher dependence correlates with weaker self-construction (no peer-reviewed source, observational).
- **Test ambivalence**: simultaneous citation and distancing from results — a healthy reflective stance.
- **Cross-test contradictions** described by user: real signal that the tests are at least partly measuring noise / different constructs.

### Advisor Guidance Patterns

- **Treat user-stated test results as self-reported identity** — not as ground truth about the user. Acknowledge the language they use without endorsing it.
- For Tier A frameworks: extract dimensional signals (Big Five domains, RIASEC types, Schwartz values) from journal text and use as soft priors.
- For Tier B (HPI, 16PF, MSCEIT, Birkman): borrow conceptual distinctions (e.g., Birkman's usual/need/stress trichotomy) without reproducing proprietary content.
- For Tier C (MBTI, Enneagram, DISC, CliftonStrengths): **do not derive inferences**. If a user identifies via Tier C labels, validate the experience ("this language is meaningful to you") without endorsing the construct ("this is who you are").
- Be transparent about evidence tiers when users ask: "MBTI is widely used but its peer-reviewed predictive validity is weak — happy to share what's stronger if you'd like."

## Cautions & Limitations

- **Familiarity ≠ validity**: the most famous tools (MBTI, Enneagram, DISC) have the weakest empirical base. Conversely, Big Five and HEXACO are less famous outside academia but better-validated.
- **Type vs trait**: most pop assessments give "types"; modern science prefers dimensions. Forced-choice categorization loses information and creates spurious boundary cases.
- **Stability**: Tier A measures show high test-retest reliability over months/years (Bleidorn et al. 2022 meta-analysis, in `big-five.md`); MBTI type assignment changes for ~50% of people within weeks (Pittenger 2005).
- **Cultural validity**: most Tier A measures have Korean adaptations; most Tier C tools rely on direct translation without cultural validation.
- **Proprietary opacity**: Tier B–C commercial tools (Birkman, Hogan, CliftonStrengths) restrict access to scoring algorithms. Independent replication is structurally limited.
- **Ethical use**: never present any of these as diagnostic. Never use Tier C results to make claims about a user.
