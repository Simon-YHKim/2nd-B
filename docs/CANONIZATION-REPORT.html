2 정본 모델 — 별자리 3-레이어

A · 입력

북두칠성 7 도메인 별 (홈에 보이는 별)

커리어 · 재정 · 성장 · 관계 · 건강 · 오락 · 담아내기. 쉬운 삶 데이터를 넣는 곳. 각 별: 입력 → 출력 + 리스트업(편집 가능 + 카테고리 + 태그).

▼
B · 검증

심리구인 (홈에 안 보임 · 숨은 추론 레이어)

Big Five · 내러티브 · 애착 · SDT/VIA · 가능자아 · 순간변동. stars.ts의 기존 7축을 "별"→"검증틀"로 재분류. 도메인 데이터가 구인을 삼각측량 → 검증된 초상(C8).

▼
C · 출력

북극성 (Polaris)

7 도메인 종합 → 실시간 페르소나(역할/모자) + 성향·강점 요약. 직접 입력 X. propose→ratify로만 변경. "Soul Core" 명칭 폐기.

밝기 정직성 규칙 — 홈의 별 빛 = "내가 얼마나 넣었나"(도메인 커버리지). 페르소나 주장 세기 = "얼마나 검증됐나"(숨은 구인 신뢰도). 이 둘을 섞지 않는다. 데이터 적은데 북극성만 밝게 빛나면 거짓말이 된다.
