2nd-Brain 화면 명세서

세상을 아는 AI를 넘어 ‘나를 아는’ 두 번째 뇌. 담을수록 별이 밝아지고, 일곱 별을 모으면 북극성이 켜집니다. 프로토타입의 전체 화면과 각 버튼의 동작을 정리했습니다.

3-레이어 모델

A 입력담기·연동·통화 녹음으로 원자료를 모으고 → B 검증Big Five·애착·가치관·동기·강점으로 교차 확인 → C 종합북극성 문장·IDEN으로 응축.

별 = 삶의 도메인

홈의 7별은 실제 북두칠성(큰곰자리) 좌표로 배치 — 커리어·재정·관계·성장·건강·휴식·담아내기. 포인터별이 가리키는 북극성은 7별의 종합입니다.

정직한 밝기

아는 척하지 않아요. 별빛(얼마나 담겼나·커버리지)과 확신(얼마나 검증됐나·신뢰도)을 나눠 보여주고, 어두운 별은 어둡다고 말합니다.

L1 흐릿L2 보임L3 또렷별빛 ≠ 확신

불변식 (제품 원칙)

① 모든 추정은 나의 비준으로만 별에 반영 ② 온디바이스 우선·원문 미저장 ③ 분석은 비차단(백그라운드 도크) ④ 열람·삭제·철회의 데이터 주권.
