2nd-Brain — 별자리 시스템 구조

단일 비유는 별자리 하나. (하늘·땅·흙·동반자 같은 다른 비유는 폐기.) 7별 = 입력 도메인, 북극성 = 종합 출력.

결정된 것 3

① 입력 ↔ 출력 분리

7별 = 입력(삶의 도메인 데이터를 넣는 곳). 북극성 = 출력(7별을 종합해 보여줌). 별은 직접 넣고, 북극성은 받아 비춘다.

② 중심 = 북극성

'Core'·'Soul Core' 안 씀. 중심 명칭은 북극성 하나로 통일.

③ 밝기 = DIKW 한 사다리

L1 꺼짐L2 DataL3 InformationL4 KnowledgeL5 Wisdom

별마다 숫자 하나 = "이 도메인의 앎이 어디까지 익었나." 모든 별이 켜지면 북극성이 더 밝아짐.

7별 종합 = 실시간 페르소나 + 성향·강점 요약

세컨비(AI 비서)가 7별·북극성을 전부 읽고 돕는다 + 바깥 트렌드를 제안으로 들여온다. 기초입력(철학·4W1H)은 시작 seed — 나중에 AI가 종합해 채울 수 있다.

7별 = 입력 도메인 각 별: 입력 → 출력 + 리스트업

1. 커리어

입력 진행중·완료 프로젝트, 이력, 업무 스타일 · 출력 추천 목표·프로젝트, 업무 방식 조언

2. 재정 수동입력 먼저

입력 자산·자본·현금흐름, 재정 목표 (계좌·카드 연동은 후순위) · 출력 가계부, 재정 트렌드, 목표 조언

3. 성장 옛 '회상'

입력 연령대별(유년기·10·20·30대) 패턴 deep drill · 출력 "그때의 나" 연령대별 요약

4. 관계 제3자 주의

입력 대상(가족·친구·동료)별 생각·지향, peer2peer · 출력 관계 정리, 대상별 조언

5. 건강 생활습관만

입력 컨디션·생활습관 자기기록 (검진결과 의료 해석은 범위 밖) · 출력 요약, 트렌드, 생활 조언

6. 오락

입력 취미·경험 기록 · 출력 취미 성장 조언, 취미 추천

7. 담아내기 catch-all

입력 6별이 못 담은 환경·기타 + 사용자가 계속 추가하는 메모·데이터 · 출력 리스트업. (모든 입력은 편집 가능 + 카테고리·태그 — 종합의 '깊이'가 여기서 나옴.)

북극성 = 출력 종합

7별을 병렬로 종합해 대표 실시간 페르소나(들)을 만든다. 각 페르소나 요약 + 내 성향·장단점·강점에 집중하는 법·조언. 밝기 = 7별 종합. 바뀔 땐 propose→ratify(AI가 먼저 산출→합의→승인)로만 확정. 페르소나(여럿) = 메모의 '모자(역할)'.

짚을 점 설계 시

검증 깊이

쉬운 삶 데이터로 받되, 북극성 종합이 검증된 틀로 근거를 대야 그냥 '인생 대시보드'와 차별됨(C8 큐레이터). 차별점 = "삶 데이터 → 검증된 나의 초상".

건강 = 의료선

검진결과 해석·의료성 조언은 우리가 피하는 범주. 생활습관·컨디션 자기기록까지만.

관계 = 제3자

실명 타인(가족·친구) 분석은 제3자 개인정보(PIPA). 범위·동의 설계 필요.

재정 = 연동 무게

계좌·카드 연동 = 오픈뱅킹급 + 민감. 수동입력으로 시작, 연동은 한참 뒤.
