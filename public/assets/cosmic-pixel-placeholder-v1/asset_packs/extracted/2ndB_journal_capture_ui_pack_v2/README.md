# 2ndB Journal / Capture UI Pack v2

This package defines the mobile vertical UI for **오늘의 조각** (`/journal`) and **조각 담기** (`/capture`) in the Cosmic Pixel Graph Village direction.

The purpose of this pack is to make the first input moment feel like a shard entering the graph village.

- Journal uses **Momo** as the record keeper.
- Capture uses **Lulu** as the collector.
- Saved records become Tier 4 data shards in the NavGraph.
- Characters should be used as brief event cues, not permanent screen decoration.
