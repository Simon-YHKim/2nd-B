# Copy Guide — SecondB Chat

세컨비 대화 UI는 clinical voice를 피하고, 창작적 자기이해와 다음 한 걸음의 톤을 유지한다.

## 권장 표현

- 참고한 조각
- 최근 기록
- 나의 모습
- 같이 이어보면
- 다음 한 걸음
- 지금은 이렇게 보여요
- 이 조각과 가까워요

## 피해야 할 표현

- 분석 결과
- 진단
- 치료
- 상담
- mental health
- therapy
- diagnosis
- treatment
