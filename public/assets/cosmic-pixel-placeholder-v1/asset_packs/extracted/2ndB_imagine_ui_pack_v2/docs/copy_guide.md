# Copy Guide — Imagine / 공상 작업실

## Preferred terms

- Imagine → 공상 작업실
- prompt → 생각 / 공상 씨앗
- result → 공상 조각
- output → 펼친 장면
- recommendation → 다음 한 걸음
- save → 마을에 저장
- source/context → 참고한 조각

## Main copy

Header:

```txt
공상 작업실
아직 말이 안 된 생각도 장면으로 펼쳐봐요.
```

Input placeholder:

```txt
예: 밤빛 조각마을의 첫 장면을 만들고 싶어
```

Generating:

```txt
벨라가 조각을 펼치는 중
관련 기록과 공상 조각을 살펴보고 있어요.
```

Save:

```txt
마을에 공상 조각으로 저장
```

Saved:

```txt
공상 조각이 그래프에 연결됐어요.
```

## Avoid

- 치료, 치유, 상담, 정신건강
- 분석 결과
- 진단
- 평가/점수
- 확정적인 자기 규정

## Prefer

- 같이 그려보면
- 장면으로 펼쳐볼게요
- 다음 한 걸음으로 접어볼게요
- 아직 말이 안 되어도 괜찮아요
