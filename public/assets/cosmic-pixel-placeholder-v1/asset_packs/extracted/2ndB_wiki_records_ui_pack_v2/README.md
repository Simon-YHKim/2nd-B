# 2ndB Wiki & Records UI Pack v2

모바일 세로 화면용 지식 창고(`/wiki`) 및 조각 기록 브라우저 UI 자산 패키지입니다.

이 패키지는 Journal/Capture에서 들어온 조각이 축적되고, 사용자가 다시 검색하고, 위키 페이지와 기록 상세에서 그래프 또는 세컨비로 이어지는 흐름을 구현하기 위한 기준 자산입니다.

## Core language

- `/wiki` → 지식 창고
- records → 조각 기록
- wiki_pages → 위키 페이지
- references / sources → 참고한 조각 / 원본 조각
- open in graph → 그래프에서 보기
- ask Jarvis / 2ndB → 세컨비에게 묻기

## Intended use

Place under:

```txt
public/assets/cosmic-pixel-v2/wiki-records/
```

Use SVGs as visual references or direct `<img>` assets. Refactor to inline SVG only if token-level recoloring is required.
