# Copy guide

## Preferred names

- Wiki → 지식 창고
- Records → 조각 기록
- Source → 원본 조각
- References → 참고한 조각
- Open in graph → 그래프에서 보기
- Ask AI → 세컨비에게 묻기

## Recommended copy

- 저장된 조각을 다시 찾아요.
- 이 페이지는 여러 조각이 모여 만들어졌어요.
- 이 조각은 이런 길과 이어져 있어요.
- 그래프에서 연결을 볼까요?
- 이 페이지로 세컨비에게 물어볼까요?
- 아직 창고가 조용해요.

## Avoid

- 분석했습니다
- 진단했습니다
- 치료, 치유, 상담, 정신건강
- 데이터베이스, RAG, embedding, vector, retrieval in user-facing copy
- 너무 게임식 보상 표현
