# Copy Guide — Core Brain / 나의 중심

## Preferred terms

- Core Brain → 나의 중심
- domain → 동네 / 영역
- persona / mask → 나의 모습
- data point → 조각
- evidence → 이걸 만든 조각들
- recommendation → 다음 한 걸음
- analysis → 살펴보기

## Tone

The screen should feel like a friendly map, not a report card.

Good:

```txt
최근 기록들이 공상 작업실과 연결됐어요.
지금의 기록에서는 “만드는 나”가 자주 보여요.
이 조각들이 이 연결을 만들었어요.
다음 한 걸음으로 줄여볼게요.
```

Avoid:

```txt
당신은 이런 사람입니다.
진단 결과입니다.
치료/상담/정신건강 관련 wording.
높음/낮음만 나열하는 점수판.
```

## Suggested microcopy

Header subtitle:

```txt
요즘 자주 이어지는 조각들을 살펴봐요.
```

Bright connection:

```txt
최근 기록들이 아이디어를 구체화하는 쪽으로 이어졌어요.
```

Evidence drawer:

```txt
세컨비와 나의 중심이 참고한 기록이에요.
```

Empty state:

```txt
아직 중심이 작아요.
오늘의 조각을 하나 남기면 세컨비가 연결을 찾아볼 수 있어요.
```
