# 2ndB Onboarding / First Run UI Pack v2

This pack defines the first-run onboarding and first-shard experience for the Cosmic Pixel Graph Village direction.

## Purpose

The user should quickly understand:

1. 2ndB turns thoughts and records into “조각”.
2. Those shards connect into a graph village.
3. SecondB answers by referring to the user's own shards.
4. The first useful action is saving one shard.

## Recommended install path

```txt
public/assets/cosmic-pixel-v2/onboarding/
```

## Key files

- `preview.html`: asset preview
- `components/OnboardingFlowSkeleton.tsx`: implementation reference
- `docs/implementation_order_for_coding_llm.md`: copy/paste handoff for coding LLM
- `docs/copy_guide.md`: approved onboarding copy
- `docs/data_contract.md`: suggested state/payload types

## Design principles

- Dark cosmic graph village, not cafe palette.
- Pixel-inspired but not childish.
- Not a farming game.
- Explain the app through shards, graph, SecondB, and first action.
- Avoid clinical language.
