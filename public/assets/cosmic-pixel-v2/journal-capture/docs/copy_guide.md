# Copy Guide — Journal / Capture

## Preferred user-facing terms
- Journal → 오늘의 조각
- Capture → 조각 담기
- Data → 조각
- Save → 마을에 조각으로 저장
- Link imported → 새 조각을 가져왔어요
- Record stored → 조각을 잘 보관했어요

## Tone
Use short, warm, non-clinical copy.

Good:
- “한 문장이어도 충분해요.”
- “모모가 조심히 보관해둘게요.”
- “루루가 새 조각을 가져왔어요.”
- “이 조각은 나중에 다른 길과 이어질 수 있어요.”

Avoid:
- 분석했습니다
- 진단했습니다
- 치료/치유/상담 관련 표현
- 과한 칭찬이나 키즈앱 말투
- 게임 보상처럼 느껴지는 점수 과다 노출
