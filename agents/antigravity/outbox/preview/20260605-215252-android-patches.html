1. Hardware BackHandler for PremiumBottomSheet

File: src/components/premium/feedback.tsx

Finding: The bottom sheet was implemented as a plain View instead of a Modal, causing the hardware back button to pop the navigation route instead of closing the sheet.

Fix: Added a useEffect hook with BackHandler.addEventListener to intercept the back press and trigger onClose().

useEffect(() => {
  if (!visible) return;
  const sub = BackHandler.addEventListener("hardwareBackPress", () => {
    onClose();
    return true;
  });
  return () => sub.remove();
}, [visible, onClose]);

Status: Applied
